# 切线冲刺

《切线冲刺》是一款移动端优先的竖屏三车道跑酷游戏。玩家驾驶未来感悬浮摩托，在通向远处城市的海上霓虹大桥上高速前进，通过左右变道躲避障碍、收集道具、维持护盾和加速状态，挑战更远距离和更高分数。

项目当前定位是浏览器游戏原型到可发布版本之间的持续打磨阶段：核心循环已经跑通，正在重点优化视角、资产、动画、反馈和移动端体验。

## 当前玩法

- 竖屏 390x844 设计画布，优先服务移动端。
- 伪 3D 透视跑道，三股主车道加两侧缓冲带。
- 主角是悬浮摩托序列帧动画，会根据所在车道朝向道路消失点微旋转。
- 障碍物和道具沿道路投影运动，带近大远小、透明度和深度变化。
- 道具包括能量、护盾和加速。
- 障碍物包括未来科技风路障、能量拒马、损坏载具和危险能源箱。
- 支持普通模式和 `?debug=1` 调试模式。

## 技术栈

- Phaser 3：游戏场景、资源加载、动画、渲染循环。
- TypeScript：核心逻辑、状态、投影和测试。
- Vite：开发服务器和生产构建。
- Vitest：状态、输入、碰撞、投影、资源清单等自动化测试。
- DOM Overlay：HUD、暂停、结算、调试面板等界面层。

## 开发环境

安装依赖：

```bash
npm install
```

启动开发服务器：

```bash
npm run dev
```

普通预览：

```text
http://localhost:5173/
```

调试预览：

```text
http://localhost:5173/?debug=1
```

调试模式会提供大量初始护盾，便于长时间观察跑道、障碍物、道具和动画效果。

## 输入方式

移动端：

- 左滑：向左变道。
- 右滑：向右变道。
- 点击：开始或确认。

PC 调试：

- 鼠标按下并左拖/右拖：模拟移动端变道手势。
- `A` / 左方向键：向左变道。
- `D` / 右方向键：向右变道。
- `Space` / `Esc`：暂停或恢复。
- `Enter`：开始或确认。
- `R`：重开。
- `F2`：显示或隐藏调试面板。

早期设计曾包含跳跃和下滑，后来为了匹配伪 3D 表现力和移动端手感，已收敛为左右变道为主的核心操作。

## 开发原则

- 移动端优先，PC 端服务调试和预览。
- 先保证核心循环和手感，再逐步提高资产、动画和特效质量。
- 游戏状态、输入、投影、碰撞和渲染尽量分层，方便测试和调参。
- 视觉改动要尽量收敛到资源清单、视觉常量和工厂类，避免把美术调参散落到玩法逻辑里。
- 每次重要迭代都运行测试和构建，并使用中文提交日志。
