# 《切线冲刺》伪 3D 体验升级设计记录

日期：2026-05-02

## 目标

本阶段目标是把主游戏画面从平面 2D 跑道升级为伪 3D 透视跑道，让背景图可见、赛道有纵深、障碍和道具具有远近感，同时保持现有三轨核心玩法不变。

## 范围

包含：

- 伪 3D 透视投影器。
- 梯形跑道和向远处汇聚的轨道线。
- 障碍和道具随 y 轴位置缩放、淡入和排序。
- 玩家位置使用同一套透视投影。
- `?debug=1` 页面显示玩家和障碍碰撞盒。
- 图片主角模式隐藏矩形拖尾，避免视觉误导。

不包含：

- 真 3D / Three.js。
- 新玩法、新轨道数或新输入。
- 新角色动画、粒子特效、音效或菜单功能。

## 架构原则

- 玩法仍然只使用 `lane`、`y` 和碰撞盒。
- 渲染层通过 `PerspectiveProjector` 把 lane 和 y 转成屏幕 x、scale、alpha 和 depth。
- 碰撞盒仍独立存在，不从图片视觉边界推导。
- debug hitbox overlay 只在 `?debug=1` 下启用。

## 后续讨论点

- 换轨时是否需要更强的轨道吸附反馈。
- 跳跃/滑铲是否加入专门动画帧或 squash/stretch。
- 疾跑时是否增强地面流动、残影和屏幕边缘速度线。
- 是否重生成偏暗的护盾资产。
